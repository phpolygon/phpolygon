// Features

Everything a game needs.
Nothing a web app has.

PHPolygon ships a complete, battle-tested set of engine systems. PHP 8.x Attributes drive serialization automatically — no #[Property] boilerplate, ever.

🧩

Hybrid ECS

Entities as PHP objects. Components with lifecycle hooks (onAttach, onUpdate, onDetach). Systems for cross-entity logic. Clear boundaries, always.

#️⃣

Attribute-driven serialization

#[Property], #[Range], #[Hidden], #[Serializable] on component fields. Reflection handles the rest — zero manual JSON mapping.

🏗️

Scene system

PHP-canonical scene definitions with fluent SceneBuilder. SceneManager with single/additive loading, parent-child hierarchy, and persistent entities across loads.

🔄

PHP ↔ JSON transpiler

Bidirectional conversion for the editor pipeline. PHP is always the source of truth. JSON is the intermediate format for tooling. Full roundtrip fidelity guaranteed.

⚡

Physics 2D

RigidBody2D, BoxCollider2D, AABB collision detection and response, raycasting, trigger and collision events — pure PHP, no external dependency.

🎮

Action-based input

Named bindings (jump, shoot, move_x) instead of raw key codes. Axis support for analog movement. Fully remappable at runtime.

🎨

NanoVG renderer

Camera2D, TextureManager, SpriteSheet — 2D rendering via NanoVG over OpenGL 4.1. Hardware-accelerated vector shapes, text, and sprites at native speed.

🔧

Editor infrastructure

Inspector metadata via Reflection, component registry with category grouping, SceneDocument with full undo/redo, command bus for all editor operations.

🔊

Pluggable audio

AudioBackendInterface with AudioSource component. Backend-agnostic by design — swap the implementation without touching a single line of game code.

MainMenu.php — scene definition 
class MainMenu extends Scene
{
    public function build(SceneBuilder $b): void
    {
        $b->entity('Camera')
            ->with(new Transform2D())
            ->with(new Camera2DComponent());

        $b->entity('Player')
            ->with(new Transform2D(
                position: new Vec2(100, 200)
            ))
            ->child('Weapon')
                ->with(new SpriteRenderer(
                    textureId: 'sword.png'
                ));
    }
}
game.php — hello world 


use PHPolygon\Engine;
use PHPolygon\EngineConfig;
use PHPolygon\Component\Transform2D;
use PHPolygon\Component\SpriteRenderer;
use PHPolygon\Math\Vec2;

$engine = new Engine(new EngineConfig(
    title: 'My Game',
    width: 1280,
    height: 720,
    vsync: true,
    targetTickRate: 60.0,
));

$engine->onInit(function ($engine) {
    $player = $engine->world->createEntity();
    $player->attach(new Transform2D(
        position: new Vec2(640, 360)
    ));
    $player->attach(new SpriteRenderer(
        textureId: 'player.png'
    ));
});

$engine->onUpdate(function ($engine, float $dt) {
    // game logic here, $dt = delta time
});

$engine->run();
